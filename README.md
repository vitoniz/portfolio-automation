# Invoice Automation — Live Demo

A live, interactive demo of a Python automation that turns raw spreadsheet
data into formatted PDF invoices. Try it directly in the browser — no
install needed.

**Live site:** https://vitoniz.github.io/portfolio-automation/

## What this shows

This page demonstrates the same automation as the
[`generate_invoices.py`](https://github.com/vitoniz/portfolio-automation)
script, rebuilt as an interactive walkthrough:

- **Before / after** — a raw spreadsheet row next to the polished invoice
  it becomes
- **Live demo** — type in a client name and line items, click *Run
  generate_invoices.py*, and watch a simulated terminal process the data
  before rendering the finished invoice, using your own numbers
- **How it works** — a plain-language breakdown of the three steps the
  real Python script performs
- **Tech stack** — Python, pandas, reportlab

## Why a demo page instead of just the script

Most people evaluating a freelancer can't run a `.py` file to see what it
does. This page lets anyone feel the automation working in under a
minute, with zero setup — the same reason a landing page beats a plain
Word doc.

## Tech used

- Plain HTML, CSS, and JavaScript (no framework, no build step)
- Hosted for free with GitHub Pages
